Актуальный код находится в файле script.py

Пошаговая инструкция по созданию инсталлятора и установки программы.

1. Внести все необходимые изменения в файл script.py или kvu.py.

2. Поместить script.py, setup.py, kvu.py в отдельную директорию.

3. Перейти из консоли в эту директорию и выполнить команду:
	 python setup.py py2exe 
(Требуется установленный py2exe модуль)

4. Скопировать директории jdcal, et_xmlfile и openpyxl, так же тестовые .xlsx файлы
в появившуюся директорию build.

5. Далее запускаем программу Inno Setup Compiler 5
	File -> New -> Next
	Вводим желаемое название, версию
	Next -> Next 
	В Application main executable выбираем .exe файл, появившийся в директории dist.
	Чуть ниже нажимаем Add folder и выбираем две образовавшиеся папки build и dist.
	Next -> Next -> Next -> Next
	В Custom comp output folder выбираем папку, в которую поместится установочный .exe файл
	Comp output base file name меняем с setup на любое другое, так как для запуска любого setup.exe
	будут дополнительные проверки на различные библиотеки и могут возникнуть конфликты или же ошибки.
	Next -> Next -> Finish 
	Далее соглашаемся на компиляцию и отказываемся от сохранения.
	После компиляции файл будет лежать в папке, которую указали при настройке.
